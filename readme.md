# Pokemon Team Builder

Description: a basic pokemon team builder that allows you to select 6 pokemon for your team and evaluate your team's type weaknesses using HTML, CSS, and JavaScript


# Technologies Used

- HTML5
- CSS3
- JavaScript
- PokeAPI
